`blode-ui` is a component library and documentation platform built with Next.js 16, React 19, Tailwind CSS v4, and Radix UI.

## Highlights

- Accessible, composable components following shadcn-style registry patterns.
- Tailwind v4 design tokens with flexible theming and dark mode support.
- Registry build pipeline that outputs installable JSON artifacts.
- Docs powered by MDX and content collections.

## Documentation

API docs, guides, and component examples are available at:

- https://ui.blode.co/docs

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Common Commands

```bash
npm run dev         # Start dev server (Turbopack)
npm run build       # Build docs, registry, and Next.js app
npm run typecheck   # Build docs + run TypeScript checks
npm run build:registry
npm run check
npm run fix
```
