Blode UI is an opinionated [shadcn/ui](https://ui.shadcn.com) component registry built by [Matthew Blode](https://matthewblode.com), with a focus on good taste, care, and craft. Open source. Open code.

## Quick Start

```bash
npx shadcn@latest init
```

Add the Blode registry and install your first component:

```bash
npx shadcn@latest registry add @blode=https://ui.blode.co/r/{name}.json
npx shadcn@latest add @blode/button
```

See the full guide at [Installation](/docs/installation).

## Registry

The shadcn registry is hosted at [ui.blode.co](https://ui.blode.co).
