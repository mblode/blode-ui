Blode UI is the official design system and React component library that powers [blode.co](https://blode.co). Built on [shadcn/ui](https://ui.shadcn.com) and [Base UI](https://base-ui.com) primitives, it provides a comprehensive set of beautiful, accessible, and customizable components that create a consistent experience across all Fingertip products.

## Key features

- **Cohesive design system**: A unified design language ensuring consistency across the Fingertip ecosystem
- **Accessible components**: Built on Base UI primitives for robust accessibility in all Fingertip interfaces
- **Customizable framework**: Adaptable components that maintain brand consistency while allowing for flexibility
- **TypeScript ready**: Full TypeScript support with comprehensive types for reliable development
- **Variable font system**: Uses the Glide variable font for modern, responsive typography across all touchpoints

Blode UI is the foundation of blode.co's interface, creating professional, polished experiences without sacrificing flexibility or performance. Developers working on Fingertip products can easily integrate these components into their React applications to maintain design consistency.

## About blode.co

Blode UI is the design system and component library that powers [blode.co](https://blode.co), an all-in-one business platform for creators, freelancers, and businesses. This UI library ensures consistent, accessible, and beautiful interfaces across the entire Fingertip ecosystem. The platform enables users to manage everything through a single link for just $8/month, replacing multiple tools costing $200+ monthly. By utilizing this design system, blode.co delivers a professional user experience that allows customers to sell products, book clients, and connect with their audience without requiring coding skills.
