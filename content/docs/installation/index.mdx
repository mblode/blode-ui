<Callout>

**Note:** Blode UI is a third-party shadcn registry hosted at [ui.blode.co](https://ui.blode.co). The setup flow is the same as [shadcn/ui](https://ui.shadcn.com/docs/installation/) with one extra step: add the registry namespace.

</Callout>

<Steps>

### Create or initialize a project

For a new project, run:

```bash
npx shadcn@latest create
```

For an existing project, run:

```bash
npx shadcn@latest init
```

### Add the Blode registry

```bash
npx shadcn@latest registry add @blode=https://ui.blode.co/r/{name}.json
```

### Add a component

```bash
npx shadcn@latest add @blode/button
```

### Import component

The command above will add the `Button` component to your project. You can then import it like this:

```tsx {1,6} showLineNumbers
import { Button } from "@/components/ui/button";

export default function Home() {
  return <Button>Click me</Button>;
}
```

</Steps>
