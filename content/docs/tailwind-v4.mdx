Blode UI now supports Tailwind v4 and React 19 by default.

## How to use Tailwind v4

You can start by going to any component and following the instructions. Try the [Button documentation](/docs/components/button) for example.

## How to upgrade your project to Tailwind v4

If you would like to upgrade your project to Tailwind v4, follow the [shadcn/ui Tailwind v4 guide](https://ui.shadcn.com/docs/tailwind-v4).
