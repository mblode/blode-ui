## Props

| Prop          | Type     | Default | Description                        |
| ------------- | -------- | ------- | ---------------------------------- |
| `size`        | `number` | 48      | The size of the spinner in pixels. |
| `strokeWidth` | `number` | 2       | The width of the spinner stroke.   |
