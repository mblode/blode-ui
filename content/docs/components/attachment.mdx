## Usage

```tsx
import {
  Attachment,
  AttachmentAction,
  AttachmentActions,
  AttachmentContent,
  AttachmentDescription,
  AttachmentMedia,
  AttachmentTitle,
} from "@/components/ui/attachment";
```

```tsx
<Attachment>
  <AttachmentMedia>
    <FileTextIcon />
  </AttachmentMedia>
  <AttachmentContent>
    <AttachmentTitle>report.pdf</AttachmentTitle>
    <AttachmentDescription>PDF Â· 2.4 MB</AttachmentDescription>
  </AttachmentContent>
  <AttachmentActions>
    <AttachmentAction aria-label="Remove report.pdf">
      <XIcon />
    </AttachmentAction>
  </AttachmentActions>
</Attachment>
```

## Examples

### States

Set `state` to `idle`, `uploading`, `processing`, `error`, or `done` to reflect the upload lifecycle.

<ComponentPreview
  name="attachment-states"
  description="Idle, uploading, processing, error, and done states"
/>

### Sizes

Use the `size` prop to switch between `default`, `sm`, and `xs`.

<ComponentPreview name="attachment-sizes" description="Default, small, and extra small sizes" />

### Image

Set `variant="image"` on `AttachmentMedia` to render an image preview.

<ComponentPreview
  name="attachment-image"
  description="Vertical image attachments with a full-card link"
/>

### Group

Wrap attachments in `AttachmentGroup` for a horizontally scrollable row.

<ComponentPreview name="attachment-group" description="A scrollable row of file attachments" />

### Trigger

Use `AttachmentTrigger` to make the whole card open a dialog while actions stay clickable above it.

  name="attachment-trigger"
